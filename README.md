# python-package

`python-package` determines the current day of the week.


## CLI instructions

```
day_of_the_week -h
```

```
usage: day_of_the_week [-h] [--day] [--info]

optional arguments:
  -h, --help   Show this help message and exit
  --day      Prints current weekday.
  --info       Returns info about package.
  ```
